🔄 Sincronizzazione Supabase

Inserisci le credenziali del tuo progetto Supabase e una chiave di sync (una parola segreta qualsiasi). Usa la stessa chiave su tutti i dispositivi per condividere i dati in tempo reale.
Come configurare Supabase (una volta sola):
1. Vai su supabase.com → crea progetto gratuito
2. SQL Editor → esegui:
CREATE TABLE agenda_sync (id TEXT PRIMARY KEY, data JSONB NOT NULL, updated_at TIMESTAMPTZ DEFAULT NOW());
ALTER TABLE agenda_sync ENABLE ROW LEVEL SECURITY;
CREATE POLICY "open" ON agenda_sync FOR ALL USING (true) WITH CHECK (true); 3. Settings → API → copia URL e anon key qui sopra
