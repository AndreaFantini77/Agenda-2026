📱 Installa su iPhone / Desktop

Salva questo file HTML e seguì i passi per installarlo come app.

iPhone / iPad
1
Salva il file agenda2026.html su iCloud Drive o AirDroppalo sul tuo iPhone
2
Aprilo con Safari (non Chrome)
3
Tocca il pulsante Condividi (□↑) in basso
4
Scorri e tocca "Aggiungi a schermata Home" → Aggiungi
5
L'app appare come icona nella home. Si apre a tutto schermo come un'app nativa.
Mac / Windows Desktop
1
Apri il file in Chrome o Edge sul desktop
2
Clicca ⋮ → "Installa Agenda 2026" (Chrome propone l'installazione come PWA)
3
L'app si apre in una finestra dedicata senza barra del browser
💡 Sincronizzazione iPhone ↔ Mac: usa il pulsante Esporta ICS per generare un file calendario. Aprilo su iPhone → Apple Calendar lo importa → iCloud lo sincronizza automaticamente su tutti i dispositivi con promemoria nativi.
